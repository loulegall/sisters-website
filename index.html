Comment épingler ce site sur ton iPhone

Pour accéder rapidement à ce site comme une application :

1

Ouvre Safari

Ouvre cette page dans Safari (pas dans Chrome ou autre navigateur)

2

Appuie sur les trois points

En bas à droite de l'écran, appuie sur les trois points horizontaux ⋯

3

Clique sur "Partager"

Dans le menu qui s'ouvre, sélectionne "Partager"

4

Clique sur "Plus"

Dans le menu de partage, fais défiler et clique sur "Plus" avec l'icône ⋯

5

Choisis "Sur l'écran d'accueil"

C'est ici que tu trouveras l'option "Sur l'écran d'accueil" - active-la si ce n'est pas déjà fait

6

Personnalise le nom

Tu peux changer le nom si tu veux (par exemple "Sisters" ou "Feel Good")

7

Ajoute à l'écran d'accueil

Appuie sur "Ajouter" en haut à droite

C'est fait ! Tu auras maintenant une icône sur ton écran d'accueil pour accéder rapidement à ce site, comme une vraie application.
